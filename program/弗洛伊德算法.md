弗洛伊德算法

是一种用于求解所有顶点对之间最短路径的动态规划算法

 核心思想

通过逐步考虑中间顶点来更新任意两点之间的最短路径

算法特点

- 适用范围：有向图或无向图（允许负权边，但不能有负权环）
    
- 时间复杂度：O(n³)，其中n是顶点数
    
- 空间复杂度：O(n²)
    
- 算法类型：动态规划

算法原理

对于每对顶点(i,j)，检查是否通过顶点k能够获得更短的路径：
dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])

 优点

- 代码简单，易于实现
    
- 能够处理负权边
    
- 一次性求出所有点对的最短路径

 缺点

- 时间复杂度高，不适合大规模图
    
- 对于稀疏图效率较低

例题：
（传送门）
#include<bits/stdc++.h>
using namespace std;
const int N = 1e4 + 5;
const int INF = 1e9;

int a[105][105]; 
int main()
{	
	ios::sync_with_stdio(false);
	cin.tie(NULL);
	
	int n, m;
	int x, y, w;
	cin >> n >> m;
	for(int i = 1; i <= n; i++){
		for(int j = 1; j <= n; j++){
			if(i == j) a[i][j] = 0;
			else a[i][j] = INF;
		}
	}
	for(int i = 1; i <= m; i++){
		cin >> x >> y >> w;
		a[x][y] = w;
		a[y][x] = w;
	}
	
	for(int k = 1; k <= n; k++){
		for(int i = 1; i <= n; i++){
			for(int j = 1; j <= n; j++){
				if(a[i][k] < INF && a[k][j] < INF)
				a[i][j] = min(a[i][j], a[i][k] + a[k][j]);
			}
		}
	}
	
	int ans = INF;
	for(int i = 1; i <= n; i++){
		for(int j = i + 1; j <= n; j++){
			int res = 0;
			
			for(int b = 1; b <= n; b++){
				for(int c = b + 1; c <= n; c++){
					if(i != b || j != c)
					res += min(a[b][c], min(a[b][i] + a[j][c], 
					a[b][j] + a[i][c]));
						
				}
			}
			ans = min(ans, res);
		}
	}
	
	cout << ans;
	return 0;
}
