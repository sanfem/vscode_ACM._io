ST算法是用于解决RMQ（区间最值问题），可以在O(N log N) 的预处理时间后，在O(1)的时间里在线处理任意去区间的最大值。
算法运用倍增的思想，设F[i,j]为A[i,i+2*j-1]区间里的最大值。
在递推时，从i=1开始，有公式：
```
F[i,j]=max(F[i,j-1],F[i+2*j-1,j-1]);
```
即是把长度为2  *  j长度的区间一分为二，他的最大值是前半段和后半段中最大值较大的一个。
有：
```cpp
int f[MAXN][MAXN],a[MAXN];
int n;
viod ST_prework(){
	for(int i=1;i<=n;i++) f[i][0]=a[i];
	int t=log(n)/log(2)+1;
	for(int j=1;j<t;j++)
		for(int c=1;c<=n-(1<<j)+1;c++)
			f[i][j]=max(f[i][j-1],f[i+(1<<(j-1))][j-1]);
}
```
在准备之后，只要求出一个K，这个K满足：
```
2*k<=r-l+1<=2*(k+1)
```
也就是说这个K是在满足在2的K次方小于区间长度前提下的最小的K，F[l,k]和F[r-2*k+1,k]也就是所求区间的左右两个半区间，直接取两个区间最大值即可。
```
k=log(r-l+1)/log(2)
```
