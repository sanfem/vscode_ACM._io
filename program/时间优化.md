4.STL的优先对列实现大小根堆：
对于默认，priority_queue,自身为大根堆，默认大数在前
可用gretaer<int> 变为小根堆
对于自定义类型，可用重载运算符<,>来实现
```
 struct Node {
    int size;
    int price;
    // 重载<运算符
    bool operator<(const Node &b) const {
        return  price<b.price;
    }
};
 
```
